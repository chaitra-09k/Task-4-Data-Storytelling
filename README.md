#  Task 4: Data Storytelling & Business Insights

##  Objective
To transform data analysis into a structured story that highlights key insights and supports decision-making.

##  Problem Statement
- Lack of visibility into sales performance across cities  
- Unclear customer behavior and contribution  
- Profit not always aligned with sales  

##  Work Done
- Performed data analysis and extracted key insights  
- Built an interactive dashboard using Tableau  
- Formulated and validated hypothesis  
- Converted findings into a clear data story  

##  Key Insights
- Sales vary significantly across cities  
- Customer segments contribute differently  
- Profit is not always proportional to sales  
- Trends fluctuate over time  

##  Hypothesis Testing
Sales performance differs across cities — validated through analysis.

##  Dashboard
An interactive Tableau dashboard was created to visualize KPIs, trends, and patterns.

##  Recommendations
- Focus on high-performing cities  
- Improve low-performing regions  
- Target key customer segments  
- Monitor profit margins  

##  Tools Used
- Python (Pandas, Matplotlib)  
- Tableau  
- SQL  

##  Outcome
Developed a complete data analysis workflow from raw data to insights and business recommendations.
